Remplacer un mot par synonyme

Si on répète plusieurs fois le même mot ou si un mot ne vous convient pas, l'éditeur PDF ONLYOFFICE vous permet de rechercher des synonymes. Retrouvez les antonymes du mot affiché aussi.

  1. Sélectionnez le mot dans votre document.
  2. Passez à l'onglet Modules complémentaireset sélectionnez
    Thésaurus.
  3. Le panneau gauche liste les synonymes et les antonymes.
  4. Cliquez sur le mot remplaçant à insérer dans votre document.
